## 🛠️ Troubleshooting & Common Issues

**❌ Script not injecting / GUI not appearing**
- Ensure your executor is running as Administrator (Windows)

- Re-download the script; a partial download may have corrupted the file

**❌ Hitbox features not working**
- Verify `hitbox_size` in your config is set above `1.0`
- Some game updates reset hitbox detection — check the changelog for patches
- Try toggling the feature off and back on mid-match

**❌ Auto shoot firing at wrong targets**
- Adjust `aim_fov` to a narrower value (e.g., `60`) to restrict targeting cone
- Enable `silent_aim = true` if you notice visible snap behavior

**❌ App crashes on iOS after sideloading**
- Re-trust the developer certificate — it may have expired
- Ensure you're using the latest iOS bundle version from the official link

**❌ Config file not saving changes**
- Run the application with write permissions enabled
